# randPiCalculation
This program is gonna calculate Pi approximation via randomness.

# How does it work?
![formula](https://render.githubusercontent.com/render/math?math=
(1)P_\square = a^2 \\
(2)P_o = \pi r^2\\ 
(3)r=\frac{a}{2} \\
\\
From \; (3) \; to \; (2) \\
P_o = \pi \frac{a^2}{2^2} = \pi \frac{P_\square}{4}\\
\Updownarrow \\
\pi = \frac{P_o\cdot 4}{P_\square}
